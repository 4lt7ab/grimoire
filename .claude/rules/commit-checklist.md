# Commit checklist

**When this applies:** Before every commit in this repo.

- [ ] Tests pass.
- [ ] `ruff check` passes.
- [ ] `ruff format` has been run.
- [ ] No migrations and code in the same commit — split them.
- [ ] Commit message is a short wizard's riddle related to the change set: 1–2 sentences, with wit and whimsy.
